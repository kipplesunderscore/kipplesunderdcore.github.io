Push Swap but mostly Push Rotate

Push Swap is a 42 School Project where you must sort a list of numbers using 2 stacks and a limited set of operations. You are given 2 stacks A and B the full list of unsorted numbers will be inserted into A, then by only using the allowed operations you must sort the list into ascending order in stack A. Your algorithm needs to return the list of operations to perform to sort the list.
